
# Premium VPN Mobile App (React Native)

Проект переписан под React Native на базе Expo.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить Metro/Expo:

```bash
npm run native -- --port 8083
```

3. Открыть приложение:
- нажать `i` для iOS-симулятора (macOS);
- нажать `a` для Android-эмулятора;
- или сканировать QR-код в приложении Expo Go.
